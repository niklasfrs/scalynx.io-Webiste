# scalynx.io Landingpage

Statische Cloudflare-Pages-fertige Website fuer `scalynx.io`.

## Inhalte
- Premium Landingpage (`index.html`)
- Rechtstexte (`impressum.html`, `datenschutz.html`, `agb.html`)
- Styles + Interaktionen (`styles.css`, `script.js`)

## Lokal testen
```bash
python3 -m http.server 8787
```
Dann `http://localhost:8787` aufrufen.
